Question Types
- definition = Asks what something is. The answer states the meaning of a term or concept. 
- mechanism = Asks how something works. The answer should explain the operation
- classification = Asks how an idea or examples are grouped or categorized. The answer should name the different categories and do a brief explanation of them. 
- cause_effect = Asks why something happened or what occured from an event. The answer should describe the output and relate it back to the cause. 
- function = Asks what the role or purpose of something is. Answer should explain the concept, process, and the purpose. 
- process = Asks for an ordered list of steps of a process over time. Answer should show how it progresses over time. 
- summary = Asks for a concise summary of a topic, major idea, or segment. The answer should combine across multiple and provide an overarching answer describing the overall topic. 
- logistics = Asks about administrative such as lecturer, teaching asssitants, scheduling, etc. 
- course_specifics = Asks about lecture series structure, references to other class materials, activites, videos, textbooks, context speciifc to how the course is taught.
